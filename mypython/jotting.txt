A SET IS A SPECIAL TYPE OF DATA STRUCTURE IN PYTHON YHAT IS USED TO STORE UNIQUE ITEMS 
TIS MEANS THAT NO ITEM CAN APPEAR MORE THAN ONCE IN A SET.
SETS ARE USEFUL WHEN U NEED TO KEEP TRACK OF CONNECTION OF TINS WITHOUT USING DUPLICATES
A TUPLE IS ANOTHER TYPE OF DATA STRUCTURE IN
PYTHON THAT ALLOWS U TO STORE A COLLECTION OF ITEMS .
HOWEVER UNLIKE LIST THEY ARE ORDERED AND UNCHANGEABLE.
FILE HANDLING IN PYTHON ALLOWS YOU TO WORK WITH FILES,
ENABLING U TO STORE AND RETRIEVE DATA EASILY.
